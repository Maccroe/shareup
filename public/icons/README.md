# Icon Placeholder Information

This directory should contain PWA icons in the following sizes:

- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

You can generate these icons from a single 512x512 source image using online tools like:

- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/favicon-converter/

For now, the app will work without icons, but they're recommended for the best PWA experience.
